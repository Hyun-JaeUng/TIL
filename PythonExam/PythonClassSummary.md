* while문은 반복 실행 횟수가 명확하지 않고, 어떤 조건을 만족하면 프로그램 종료하고자 할 때 사용



### 반복문의 제어: `break`, `contine`, `else`

* break: 반복문에서 논리적으로 반복을 종료하는 방법

* continue: 특정 조건에서 남은 명령을 건너뛰고 다음 반복문을 수행함

* else: 어떤 조건이 완전히 끝났을 때 한 번 더 실행해 주는 역할을 함

  * ``` python
    for i in range(10):
        print(i)
    else:
        print("End of Program")
    ```

  * 이처럼 if - else 뿐만 아니라 for - else도 가능함!!

### 함수

* 자주 반복되는 코드 사용이 용이해지고, 일련의 코드 블록에 이름을 붙여 정의한 것

* 호출문으로 실행할 수 있음

  * 함수(아규먼트들...)

* ```python
  def calcsum(n):
      sum = 0
      for num in range(n+1):
          sum += num
      return sum
  
  print("~ 4 =", calcsum(4)) # 실행 결과: ~4 = 10
  print("~ 10 =", calcsum(10)) # 실행 결과: ~10 = 55 
  ```

* 여기서 아규먼트는 4,10 이 되는 것이고, 매개변수는 n 임. (아규먼트는 데이터 값, r-value 느낌)

* 함수 이름 지을 때 일반적으로 소문자로 입력하고, 띄어쓰기 할 경우 언더바 이용

* 매개변수: 함수가 호출될 때 전달받고자 하는 **데이터를 저장하는 변수**

* 아규먼트: 함수를 호출하면서 전달하는 데이터 (그 값이라 생각하면 됨)

* `return` : 함수의 실행 결과를 호출원으로 전달하는 값

  * **리턴 값이 없는 함수는 자동으로 `None`이 리턴 됨!**



### 지역 변수와 전역 변수

* 함수 내부에 있는 변수는 지역 변수임

* ```python
  def kim():
      temp = "김과장의 함수"
      print(temp)
  
  def lee():
      temp = 2 ** 10
      return temp
  
  # 여기서 temp는 함수 내부에서 선언되었으므로 지역 변수임
  # 또한 함수 만드는 해당 코드들은 실제로 수행되는 것이 아니라 호출됐을 떄 사용된다. 
  ```
* ```python
  price = 1000
  def sale():
      price = 500
  sale()
  print(price)
  
  # 이 때 실행값은 1000임
  # 500은 함수 내부에서만 실행 되는 지역변수이므로, 가장 위에 있던 전역변수로 선언된 1000이 출력됨
  ```

* `chr(65)` 은 A 이후 알파벳이 순서대로 증가하고, 소문자 a부터는 `chr(97)` 임
* else - pass 인 부분은 데드코드이기때문에 가독성이 필요할 때만 사용해야 함
* `type(n)` : n의 type이 무엇인지 알아내는 함수 
  * 결과값: int, float, list, tuple, str 등등
* 함수의 return은 마지막에 1번하는 것이 좋음
* 1번만 사용한다면 굳이 변수 사용 필요하지 않음
  *  2번 이상 사용 시 변수 선언하여 편리성을 높여라!

* ```python
  def sale():

* ```python
  price = 1000
  
  def sale():
      global price        # 이 함수내에서 값을 할당하는 price 는 전역 변수라고 설정하는 결과가 됨!
      price = 500         # 전역변수 price 의 값을 변경하게 된다.
      print("sale", price)
  
  sale()
  print("global", price)
  ```

* `global price`을 통해 함수 안쪽이지만 전역변수로 설정함

* 키워드 아규먼트: print의 seq이나 end 같은 것으로 상황에 따라 설정해도 되고, 안해도 되는 것

### 자료구조

* 정보(데이터)를 메모리에 효율적으로 저장하고 사용하는 방법을 정의한 것으로, **데이터를 관리하는 방식**
* 대용량일수록 메모리에 빨리 저장하고 빠르게 검색하여 데이터의 효율적 사용을 가능하게 함
* 파이썬에서 지원하는 자료구조의 종류
  * 리스트(List), 튜플(tuple), 세트(set), 딕셔너리(dictionary), Collection 모듈
  * 이것들을 통틀어서 `API`라고 함



### 스택

* 자료구조의 핵심 개념 중 하나로 `Last In First Out(LIFO)`으로 정의 할 수 있음

* 마지막에 들어간 데이터가 가장 먼저 나오는 형태로 데이터의 저장공간을 구현하는 것

* 스택에서 데이터를 저장하는 것을 푸시(push), 데이터를 추출하는 것을 팝(pop) 라고 함

* 파이썬에서는 리스트를 사용하여 스택을 구현할 수 있음

* ```python
  a = [1,2,3,4,5]
  a.append(10)
  a.pop() 
  # append로 추가된 a에 pop() 함수를 처음 실행하면, 가장 마지막에 저장된 10이 추출됨
  ```



### 큐

* 먼저 들어간 데이터가 먼저나오는 `FIFO`의 메모리 구조를 가지는 저장 체계
* 파이썬에서 큐를 구현하는 것은 pop()함수를 사용할 때 인덱스가 0번째인 값을 쓴다는 의미로 `pop(0)`을 사용하면 됨
* pop() 함수가 리스트의 마지막 값을 가져온다고 하면, pop(0)은 맨 처음 값을 가져온다는 뜻
* 파이썬에서 리스트를 사용해서 스택과 큐를 구현할 수 있음

###  파이썬의 내장 자료구조

* 컬렉션 : 문자열, 튜플, 리스트, 셋, 딕셔너리 
  * `len()`, `in` 사용 가능
* **시퀀스형: 문자열, 튜플, 리스트**
  * `indexing`, `slicing`, `unpacking` 가능
  * **여러 자료를 순서대로 넣는다는 뜻**
  * `s[0]`, `s[2:5]`,`s[5,11,2]` 와 같은 인덱싱과 슬라이싱이 가능 (슬라이싱에서 range 처럼 증가치 설정도 가능 )
  * `for d in 시퀀스`  사용 가능함
* 집합형: 셋
* 매핑형: 딕셔너리
* **불변: 문자열, 튜플** (immutable)
* **가변: 리스트, 셋, 딕셔너리** (mutable)
  * 리스트와 셋의 차이점: 리스트는 중복 허용, 셋은 중복 허용하지 않음
  * 딕셔너리는 값만 저장하는게 아니라 이름도 함께 저장함



### 리스트

  * 앞의 대괄호는 행에 대한 인덱스, 뒤의 대괄호는 열에 대한 인덱스로 사용

* ```python
  # 리스트 연산 예제
  list1 = [1, 2, 3, 4, 5]
  list2 = [10, 11]
  listadd = list1 + list2 # 뒤에 붙음
  print(listadd)
  listmulti = list2 * 3 # 뒤에 붙음
  print(listmulti)
  
  #listadd = [1,2,3,4,5,10,11]
  #listmulti = [10,11,10,11,10,11]
  ```

* `리스트.insert(위치, 값)` : 해당 리스트 위치에 값을 삽입

*  `list1.extend(list2)` : 리스트1에 리스트2 이어 붙여서 데이터 확장

* `list1.remove(100)`: list1 내부에 100값을 가지는 요소를 한 개 삭제

* `score.index(100)`: score 라는 리스트 안에 100이라는 값을 가진 위치를 반환

* `score.count(100)`: score라는 리스트 안에 100이라는 값을 가진 갯수를 반환

* random 모듈 import 한다면 `random.choice(list_name)` 으로 list 내부의 아무 원소나 선택하는 함수

* else - pass 문 자주쓰는 습관이 있는데, 고치면 더 깔끔하다.

* 아규먼트가 아무것도 전달 되지 않을 때 -> `len(n) == 0` 활용

* `def sumAll(*n):`  가변형 아규먼트!! 

  * 전달 받을 수 있는 아규먼트 개수에 제한이 없는 것을 말함

* ```python
  def sumAll(*n):
      sum = 0
      count = 0 # count도 유용하게 활용 가능함
      for i in n:
          if type(i) == int or type(i) == float: # 자료형이 정수거나 실수이면
              sum += i
              count+=1
  
      if len(n) == 0 or count == 0:
          sum = None
  
      return sum
  ```
